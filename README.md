# College Admission Information Portal

This project is a basic multi-page website built using HTML and CSS. It provides information about a college including courses, admission details, contact information, and a student enquiry form.

## Files Included

• index.html - Home page  
• about.html - About the college  
• admission.html - Admission process details  
• courses.html - Courses offered by the college  
• contact.html - Contact information page  
• enquiry.html - Student enquiry form  


## Features

• Simple and clean user interface  
• Navigation links to switch between pages  
• Well-structured content using lists, tables, and sections  
• Professional color theme (Navy Blue and White)  
• Responsive full-width enquiry form fields  
• Works in all modern web browsers  

## How to Use

• Download the project files  
• Open index.html in any browser  
• Navigate through the website using the menu links  

## Technologies Used

• HTML for structure  
• CSS for styling
